⚠️ Setup do Banco de Dados necessário

Execute este SQL no Supabase (SQL Editor) para criar a tabela de sites:

CREATE TABLE IF NOT EXISTS public.sites ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, name TEXT NOT NULL, description TEXT, url TEXT NOT NULL, category TEXT NOT NULL DEFAULT 'outros', thumbnail TEXT, created_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE public.sites ENABLE ROW LEVEL SECURITY; CREATE POLICY "Allow all" ON public.sites FOR ALL TO anon USING (true) WITH CHECK (true);
➕ Adicionar Site / App
Isso aparece automaticamente na Home para todo mundo.
